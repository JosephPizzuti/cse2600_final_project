# CSE 2600: Final Project
### Notes
#### - Please remember to make a separate branch before starting work!
#### - Please ensure all data made by notebooks is routed into the `data/` directory!
## Setup [1] - Installing data on to your machine
1. Download the `sleep_health_dataset.csv` file from [here](https://drive.google.com/drive/folders/1EHcZmTGN26HNx1OJFt0qZ6xLdtMPt8mW?usp=sharing).
2. Place `sleep_health_dataset.csv` into `data/`.
## Setup [2] - Getting data sets from the Notebooks
1. Once you have `sleep_health_dataset.csv` in `data/`, run all cells in the notebook `01_preprocessing.ipynb`.
2. Once you run preprocessing, run all cells in notebook `02_clustering.ipynb`.   
Doing the above will give you all the data you need to run `03_logistic_regression.ipynb` and `04_xgboost.ipynb`.
## Data Origins
The data used for this project was sourced from Kaggle, and you can find its container website [here](https://www.kaggle.com/datasets/mohankrishnathalla/sleep-health-and-daily-performance-dataset).
